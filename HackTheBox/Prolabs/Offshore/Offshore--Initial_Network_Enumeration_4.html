Dump just the IPs & eliminated the first line because it's .0

grep -Eo '[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+' pingsweep.txt | awk '{print $1}' > ip_list.txt

└─$ cat ip_list.txt

10.10.110.0

10.10.110.3

10.10.110.123

10.10.110.124

Do port scan on all the hosts we found

nmap -iL ip_list.txt -p- -oN tcp_portscan.txt

There aren't that many so we can dump them all into the same file

grep 'open' tcp_portscan.txt | cut -d '/' -f 1 | while read port; do echo $port >> open_ports.txt; done

Remove the duplicates

sort open_ports.txt | uniq -i > open_ports2.txt && rm open_ports.txt && mv open_ports2.txt open_ports.txt

└─$ cat open_ports.txt

443

80

8000

8089

Let's scan the version (-sV) and run any nmap scripts (-sC) for the ports we found open

nmap -sV -p $(cat open_ports.txt | tr '\n' ',') -iL ip_list.txt -oN service_scan.txt
